# Pro-Skills Clone 🚀

A pixel-perfect clone of the **Pro-Skills** landing page, built from a Figma design using pure HTML and CSS.

## 🔗 Live Demo
[View Live]()



## 🛠️ Built With
- HTML5
- CSS3
- Font Awesome (icons)

## ✨ Features
- Responsive Navbar with Login & Sign In buttons
- Hero Section with animated elements (rocket, trophy, decorative dots)
- Stats Section (1000+ Courses, 5000+ Students, 200+ Trainers)
- Hover effects on navigation links and buttons
- Figma design to HTML/CSS conversion
